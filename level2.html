Games Development - Level 2

Course Overview

An introductory course designed for ages 16-18. This college-level programme covers the fundamentals of games development including game design principles, basic programming, and using industry-standard tools to create your first games.

Qualifications

  • Level 2 Qualification in Games Development
  • Focus on core skills and concepts

Assessment Methods

  • Practical projects
  • Written assignments
  • Portfolio development

Course Details

Level: 2
Length: 1 year
Study Mode: Full-Time
Start Date: 1st September 2026
Location: Rye Hill Campus
Age Range: 16-18 College

Trips and Activities

Throughout the course, you'll have opportunities to attend industry events, visit game studios, and participate in game jams.
